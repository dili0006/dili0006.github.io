Five html tags and Description

" p " p tag is used to make paragraphs in a html document.
"Header" Header tag is the heading tag in a html document .There are 6 heading tags : h1,h2,h3,h4,h5, and h6.
"Title" Title tag is used to give titles in an html document.
"a href" an anchor tag is used to link pages .
"block qoute" this tag is used to cite qoutes.
"img" Image tag is used to add images.
